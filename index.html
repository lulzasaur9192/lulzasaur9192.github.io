Quick Start

Call any endpoint with a simple GET request. All APIs return JSON with consistent structure.

// Search for Pokemon cards GET /tcgplayer/search?query=charizard&limit=5 // Search for guitars on Reverb GET /reverb/search?query=fender+stratocaster&limit=10 // Find plumbers in Austin GET /thumbtack/search?query=plumbers&location=austin-tx&limit=10

Or use the MCP server directly in Claude, Cursor, or any MCP-compatible AI tool:

npm install -g @lulzasaur9192/marketplace-search-mcp
